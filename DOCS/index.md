### Rotas da API
1. Login
2. Logout
3. Usuario
    * Avatar
4. Deposito
5. Produtos
    * Avatar
    * Estoque
    * Preços
6. Entidades
    * Avatar
7. Documentos
    * Produtos
    * Fechar documento
    * Abrir documento


Features:
1. Os dados não são excluidos(exceto o token), apenas marcados como deletados.
2. Todas que é criado leva o id do usuario que o fez.
3. transações de update e delete gravam logs.
